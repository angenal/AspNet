EO.Pdf HTML to PDF converter can convert only a portion of an HTML page. To convert a portion of the page, set HtmlToPdf.Options.VisibleElementIds to a list of element IDs you wish to convert. The following sample calls ConvertUrl to convert our home page, but only convert the main content by setting HtmlToPdf.Options.VisibleElementIds to "main_content_div". Here "main_content_div" is a DIV in our home page that contains the main content.
