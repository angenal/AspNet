EO.Pdf HTML to PDF converter can automatically generate bookmarks based on HTML header elements (H1 to H6). It automatically builds parent/child relationship based on the location and level of the header elements. For example, all H2 elements in between two H1 elements are considered child bookmarks for the first H1 element. To use this feature, simply set EO.Pdf.HtmlToPdf.Options.AutoBookmark to true.

You can also examine the HtmlDocument object returned by the converter to create bookmark yourself based on your own application logic. See here for a working example about this feature.

Please enter any Url with header elements and then click "Run Demo" to convert it to a PDF file. Note: You may need to right click the document, then select "Show Navigation Panel Buttons" to view the generated bookmarks.
