EO.Pdf HTML to PDF converter can automatically add page header and footer. The header and footer can contain any valid HTML tags as well as "variables" that can be the page Url, page number or total page count. The following sample adds page header containing the page Url and page footer containing the current page number and total page count. Both header and footer also use <hr /> to render a horizontal line. In addition, the footer uses style="text-align:center" to center align the page numbers.
