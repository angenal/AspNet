By default, the HTML to PDF converter automatically starts the conversion as soon as all the contents of the page (such as images) are fully loaded. However, sometimes you may need to perform some additional initialization tasks with JavaScript and the conversion should not start until your initialization code is done. In this case you can manually trigger the conversion from JavaScript.
