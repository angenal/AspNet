You can set HtmlToPdf.Options.StartPageIndex and HtmlToPdf.Options.StartPosition to start render from any page position. This also means you can render multiple layers of output on the same page by calling the converter multiple times. Because EO.Pdf HTML to PDf converter renders HTML with a transparent background, the lower layer will show through unless it is being completely obscured by the layers above it. This sample demonstrates how to create a bottom layer (as watermark in this sample) and a top layer (as main content in this sample) on the same page.
