EO.Pdf HTML to PDF converter honors CSS page-break-before and page-break-after attribute. You can set either of these two attributes to always to insert a hard page break before or after any HTML element. In addition, the converter also supports page-break-inside:avoid style attribute. Applying this style to a HTML element instructs the converter not to automatically break the elements into multiple pages.
