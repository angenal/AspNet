EO.Pdf HTML to PDF converter can output the result to a physical file, a stream or a PdfDocument object. You can call ConvertHtml or ConvertUrl multiple times on the same PdfDocument object thus merging multiple Urls into the same PDF file. The new conversion can even start immediately after the previous conversion on the same page.
