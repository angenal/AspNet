EO.Pdf HTML to PDF converter can automatically repeats table header and footer when a table spans over multiple pages. To use this feature, simply use thead and tfoot instead of tr for the header row and footer row respectively.

Click "Run Demo" to see this feature in action. For demonstration purpose, The HTML being converted is hardcoded. Click "View C# Source Code" or "View VB.NET Source Code" on the toolbar for more details.
