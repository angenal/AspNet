EO.Pdf HTML to PDF converter can convert page that requires authentication. To convert such a page, simply set HtmlToPdf.Options.UserName and HtmlToPdf.Password to the correct user name and password.

The default Url below is a test page in the Web Demo application. The test page requires user name "eopdf" and password "eopdf". Any other user name or password will result in an authentication exception.

You can try out any other Url you know of that requires authentication as well. Make sure to change the UserName and Password accordingly if you try a different Url.
