This folder contains the following third-party DLL files that is 
needed to build the sample project:

ASP.NET MVC 4 Runtime DLL(s)

EO MVCToPDF supports all major ASP.NET MVC versions from MVC 3 and 
above. The MvcDemo sample project is built with ASP.NET MVC 4. This
folder contains MVC 4 runtime DLLs so that you will be able to open
and run the demo project without having MVC 4 installed. Your own
application can use any of the supported MVC versions. See here for
